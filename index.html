1. Podstawowa struktura HTML5

Każdy dokument html powinien mieć określoną strukturę.

<!DOCTYPE html>
<html lang="pl">
<head>
    <meta charset="UTF-8">
    <title>Tytuł strony w karcie przeglądarki</title>
</head>
<body>
    <header>Nagłówek strony</header>
    <main>Treść główna</main>
    <footer>Stopka z podpisem </footer>
</body>
</html>

2. Formatowanie tekstu

  • <h1> do <h6> - nagłówki (h1 najważniejszy).
  • <p> - akapit (paragraf).
  • <br> - przełamanie linii.
  • <strong> lub <b> - pogrubienie tekstu.
  • <em> lub <i> - pochylenie tekstu (kursywa).
  • <span> - element liniowy, używany do ostylowania konkretnego wyrazu.
  • <cite> - oznacza tytuły dzieł
  • <blockquote> - służy do oznaczania cytatów blokowych
  • <sup> i <sub> - indeks górny i indeks dolny
  • <code> - oznacza fragmenty codu komputerowego

3. Listy

Lista nieuporządkowana

<ul>
    <li>Element pierwszy</li>
    <li>Element drugi</li>
</ul>

Lista uporządkowana:

<ol>
    <li>Krok 1</li>
    <li>Krok 2</li>
</ol>

4. Linki i obrazy

Wstawianie grafiki z tekstem alternatywnym oraz odnośników.

<a href="https://www.tl.krakow.pl" target="_blank">Link w nowej karcie</a>

<img src="obrazek.png" alt="obrazek" height="150">
Ważne: Brak atrybutu alt w obrazku to częsty błąd, za który traci się punkty na egzaminie!

7. CSS: Selektory i sposoby dołączania

Style można dołączać na 3 sposoby: zewnętrzny (za pomocą znacznika <link>), wewnętrzny (w sekcji head <style>) oraz inline (atrybut style="").

Kluczowe selektory:

/* Selektor znacznika */
p { color: white; }

/* Selektor klasy (kropka) - używany wielokrotnie */
.czerwony-tekst { color: red; }

/* Selektor identyfikatora (hash) - unikalny, raz na stronę */
#blok-glowny { background-color: gray; }

/* Pseudoklasa stanu (np. po najechaniu myszą) */
a:hover { text-decoration: underline; }

8. CSS: Model pudełkowy

Każdy element na stronie jest pudełkiem. Marginesy decydują o ułożeniu elementów względem siebie.

  • width / height - szerokość i wysokość elementu.
  • padding - margines wewnętrzny (odstęp treści od obramowania).
  • border - obramowanie (np. border: 1px solid black;).
  • margin - margines zewnętrzny (odstęp między elementami). margin: 0 auto; centruje blok w poziomie.

9. CSS: Pozycjonowanie

Właściwość position określa sposób, w jaki element jest umiejscawiany w dokumencie HTML. Możliwe wartości to:

  • static - Wartość domyślna. Element wyświetla się w normalnym przepływie dokumentu. Atrybuty top, bottom, left i right na niego nie wpływają.
  • relative - Pozycjonowanie względne. Element jest przesuwany względem swojej domyślnej (statycznej) pozycji, ale zachowuje swoje pierwotne miejsce w przepływie strony.
  • absolute - Pozycjonowanie bezwzględne. Element jest wyjmowany z normalnego przepływu i pozycjonowany względem najbliższego rodzica, który posiada własność `position` inną niż `static`.
  • fixed - Pozycjonowanie stałe. Element jest wyjmowany z przepływu i "przyklejany" w ustalonym miejscu na ekranie względem okna przeglądarki (nie zmienia pozycji podczas scrollowania).

10. Responsywność

Projektowanie RWD (Responsive Web Design) sprawia, że witryna dostosowuje się do ekranów telefonów komórkowych.

/* Standardowe style dla komputera */
body { background-color: #1e1e24; }

/* Style aktywowane TYLKO, gdy szerokość ekranu jest mniejsza lub równa 768px */
@media (max-width: 768px) {
    body {
        background-color: #000000;
    }
    main {
        padding: 10px;
    }
}

11. CSS: Animacje

Dzięki CSS możemy płynnie zmieniać stan elementów za pomocą @keyframes.

Do animacji wykorzystujemy klatki kluczowe, w których precyzyjnie definiujemy wygląd na każdym etapie.

/* Definicja klatek kluczowych animacji */
@keyframes pulsowanie {
    0% { transform: scale(1); }
    50% { transform: scale(1.1); background-color: #00adb5; }
    100% { transform: scale(1); }
}

.element-animowany {
    /* Uruchomienie animacji: nazwa, czas trwania, powtarzanie w nieskończoność */
    animation: pulsowanie 2s infinite;
}

12. CSS: Gradienty

Gradienty w CSS pozwalają na płynne przejścia między dwoma lub wieloma kolorami. Traktowane są przez przeglądarkę jako obrazy tła, dlatego używamy dla nich właściwości background.

Gradient liniowy (Linear Gradient)

Zmienia kolory wzdłuż wyznaczonej prostej linii. Możemy określić konkretny kierunek, np. w dół, w prawo lub pod kątem.

.liniowy {
    /* Domyślne przejście od góry do dołu */
    background: linear-gradient(red, blue);
    
    /* Przejście w prawą stronę z wykorzystaniem wielu kolorów */
    background: linear-gradient(to right, red, orange, yellow, green);
    
    /* Przejście pod kątem 45 stopni */
    background: linear-gradient(45deg, #00adb5, #1e1e24);
}

Gradient promienisty (Radial Gradient)

Zmienia kolory promieniście od wyznaczonego środka na zewnątrz.

.promienisty {
    /* Domyślny gradient ze środka na zewnątrz */
    background: radial-gradient(white, black);
    
    /* Gradient w kształcie idealnego koła z ustawieniem proporcji kolorów */
    background: radial-gradient(circle, #00adb5 0%, #1e1e24 70%);
}
